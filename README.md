# nasa-with-vue

## Baby steps!

Install vue cli: `npm install -g @vue/cli`

Create new proyect: `vue create nasa-with-vue`

Add Vue plugins: `vue add unit-jest`
Add dependencies: `npm install axios`

test: `npm run test:unit`


## Project setup
```
npm install
```
